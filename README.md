# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping and purchase behavior using transactional data to uncover insights into customer segments, spending patterns, product performance, and subscription impact. The project follows an end-to-end data analytics workflow including data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard development.

The objective of this project is to generate business insights to improve customer retention, increase revenue, and optimize marketing strategies.

---

## Dataset Information
- **Total Records:** 3,900 transactions  
- **Total Columns:** 18  
- **Dataset Type:** Customer shopping and transactional data  

### Dataset Features

#### Customer Demographics
- Age
- Gender
- Location
- Subscription Status

#### Purchase Information
- Item Purchased
- Category
- Purchase Amount
- Season
- Size
- Color

#### Shopping Behavior
- Discount Applied
- Promo Code Used
- Purchase Frequency
- Review Rating
- Shipping Type

## Project Workflow

### 1. Data Cleaning and Preparation (Python)
- Loaded dataset using pandas
- Handled missing values in review rating column
- Standardized column names
- Removed duplicate and unnecessary columns
- Created new features such as age groups and purchase frequency

### 2. Exploratory Data Analysis (EDA)
- Purchase amount distribution
- Customer segmentation by age and gender
- Revenue analysis by category
- Discount vs non-discount purchase behavior
- Review rating analysis

### 3. SQL Business Analysis
SQL queries were written to solve business problems such as:
- Revenue by gender
- Revenue by category
- Subscribers vs non-subscribers analysis
- High spending customers
- Top-rated products
- Repeat customers and loyalty analysis
- Shipping type performance

### 4. Power BI Dashboard
An interactive dashboard was created to visualize:
- Total Revenue
- Total Customers
- Average Order Value
- Subscriber Percentage
- Revenue by Category
- Revenue by Age Group
- Subscribers vs Non-Subscribers
- Customer Ratings

### 5. Reporting
- Created a business insights report
- Prepared a presentation for stakeholders

---

## Key Business Insights
- Clothing and Accessories categories generate the highest revenue.
- Non-subscribers contribute more revenue, but subscribers purchase more frequently.
- Younger and middle-aged customers generate the majority of sales.
- Express shipping customers tend to have higher order values.
- Discounts increase purchase volume but reduce profit margins.

---

## Business Recommendations
- Improve subscription benefits to increase customer retention.
- Introduce loyalty programs for repeat customers.
- Optimize discount strategies by category.
- Focus marketing on high-performing categories.
- Use targeted marketing based on age group.
